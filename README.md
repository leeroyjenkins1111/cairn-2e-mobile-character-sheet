# Cairn 2e Mobile Character Sheet

Lekka, mobilna karta postaci do Cairn 2e działająca jako pojedynczy plik `index.html`.

## Aplikacja

https://leeroyjenkins1111.github.io/cairn-2e-mobile-character-sheet/

## Uruchomienie lokalne

Pobierz `index.html` i otwórz go w przeglądarce. Aplikacja nie wymaga Node.js, instalowania zależności ani połączenia z backendem.

Dane postaci są zapisywane lokalnie w przeglądarce. Warto regularnie eksportować kopię zapasową JSON.

## Publikacja

Zmiany w gałęzi `main` są wdrażane przez workflow GitHub Pages znajdujący się w `.github/workflows/deploy-pages.yml`.
